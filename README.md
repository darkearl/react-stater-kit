# react-stater-kit
It's a nice beginning for a new react project with redux.

## Demo
  https://darkearl.github.io/react-stater-kit/
## Howto

Install Dependencies:

1. NodeJS( ~> 6.2.2 )

   `$ brew update && brew install node`

2. npm( ~> 3.10.3 )

After finish install dependencies.

Then lauch it
1. `$ npm install`
2. `$ npm start`

Done.

## Support
  - Structure MVC
  - Sass/less
